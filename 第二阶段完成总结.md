**核心文件：**

**功能特性：**

- ✅ 安全等级前置过滤

- ✅ 部门权限过滤（支持部门树递归）
- ✅ 角色权限匹配
- ✅ 文档级行权限（可选）
- ✅ 过滤表达式优化器
- ✅ 权限快速验证

**安全架构：**
```
用户请求
  ↓
1. 接入层：身份认证 + 权限解析
  ↓
2. 前置验证：validate_permission() 快速拒绝
  ↓
3. 构建过滤：build_filter_expression()
  ↓
4. 向量检索：带权限过滤的相似度搜索
  ↓
5. 返回校验：二次权限验证
  ↓
6. 审计日志：全链路记录
```

**使用示例：**
```python
from app.services.vector_filter import (
    get_filter_instance,
    create_user_permission
)

# 创建用户权限上下文
permission = create_user_permission(
    user_id="user_001",
    tenant_id="tenant_001",
    dept_id="dept_001",
    role_ids=["role_admin"],
    security_level=4,
    allowed_space_ids=["space_001", "space_002"],
)

# 获取过滤器实例
vector_filter = get_filter_instance()

# 前置权限验证
if not vector_filter.validate_permission(permission, "space_001"):
    raise PermissionError("无权访问")

# 构建过滤表达式
filter_expr = vector_filter.build_filter_expression(permission, "space_001")
# 输出: 'tenant_id == "tenant_001" and security_level <= 4 and space_id == "space_001" ...'
```
